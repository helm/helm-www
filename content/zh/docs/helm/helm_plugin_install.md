## helm plugin install

安装一个或多个Helm插件

### 简介

该命令允许您通过VCS仓库url或者本地路径安装插件。

```shell
helm plugin install [options] <path|url>... [flags]
```

### 可选项

```shell
  -h, --help             help for install
      --version string   specify a version constraint. If this is not specified, the latest version is installed
```
