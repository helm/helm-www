## helm completion bash

为bash生成自动补全脚本

### 简介


为Helm生成针对于bash shell的自动补全脚本。

在当前shell会话中加载自动补全：

    source <(helm completion bash)

为每个新的会话加载自动补全，执行一次：
- Linux:

      helm completion bash > /etc/bash_completion.d/helm

- MacOS:

      helm completion bash > /usr/local/etc/bash_completion.d/helm


```shell
helm completion bash
```

### 可选项

```shell
  -h, --help   help for bash
```
