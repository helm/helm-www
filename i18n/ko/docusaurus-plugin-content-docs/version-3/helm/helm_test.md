릴리스에 대하여 테스트를 수행한다.

### 개요


test 명령어는 릴리스에 대한 테스트를 실행한다.

이 명령어가 받는 인수는 배포된 릴리스의 이름이다.
실행할 테스트는 설치된 차트에 정의되어 있다.


```
helm test [RELEASE] [flags]
```

      --logs               테스트 파드에서 로그를 덤프 (모든 테스트가 완료된 후 정리 전에 실행)
