명명된 릴리스에 대한 매니페스트 다운로드

### 개요


이 명령은 주어진 릴리스에 대해 생성된 매니페스트를 가져온다.

매니페스트는 해당 릴리스의 차트에서 생성된 Kubernetes 리소스를 YAML로 인코딩하여 표현한 것이다.
차트가 다른 차트에 의존하는 경우, 해당 리소스도 매니페스트에 포함된다.


```
helm get manifest RELEASE_NAME [flags]
```
