## helm get values

명명된 릴리스에 대한 값 파일 다운로드

### 개요


이 명령어는 주어진 릴리스에 대한 값 파일을 다운로드한다.


```
helm get values RELEASE_NAME [flags]
```

### 옵션

```
  -a, --all             모든 (계산 된) 값 덤프
  -h, --help            values 에 대한 도움말
  -o, --output format   지정한 형식으로 출력을 표시. 허용되는 값: table, json, yaml (기본값 table)
      --revision int    리비전으로 명명된 릴리스 가져오기
```
