주어진 차트에 대한 종속성을 나열

### 개요


차트에 선언된 모든 종속성을 나열한다.

이 명령은 차트 아카이브 및 차트 디렉토리를 입력으로 사용할 수 있다.
차트의 내용은 변경되지 않는다.

차트를 로드할 수 없는 경우 오류가 발생한다.


```
helm dependency list CHART [flags]
```
