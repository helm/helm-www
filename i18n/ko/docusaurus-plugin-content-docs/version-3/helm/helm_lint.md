차트에 발생 가능한 문제가 있는지 검사한다

### 개요


이 명령은 차트 경로를 받아 해당 차트가 올바른
형식(well-formed)인지 검증하는 일련의 테스트를 실행한다.

린터(linter)가 차트 설치 실패를 일으킬 사항을 발견하면
[ERROR] 메시지를 출력한다. 컨벤션이나 권장 사항에 어긋난 이슈를 발견하면
[WARNING] 메시지를 출력한다.


```
helm lint PATH [flags]
```
