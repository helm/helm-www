차트 저장소에서 로컬로 사용 가능한 차트의 정보를 업데이트한다

### 개요


업데이트 명령어는 각 차트 저장소에서 차트에 대한 최신 정보를 가져온다.
이 정보는 로컬에 캐싱되어 'helm search' 등의 명령에서 사용된다.

원하는 저장소만 지정하여 업데이트할 수도 있다.
	$ helm repo update <repo_name> ...
모든 저장소를 업데이트하려면 'helm repo update'를 사용한다.


```
helm repo update [REPO1 [REPO2 ...]] [flags]
```

### 옵션

```
      --fail-on-repo-update-fail   저장소 중 하나라도 업데이트에 실패하면 전체 업데이트 실패로 처리
  -h, --help                       update 명령어에 대한 도움말
      --timeout duration           인덱스 파일 다운로드 완료까지 대기 시간 (기본값 2m0s)
```
