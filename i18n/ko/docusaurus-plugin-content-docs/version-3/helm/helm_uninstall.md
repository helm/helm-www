릴리스 삭제

### 개요


이 명령어는 릴리스 이름을 사용하여 릴리스를 삭제한다.

차트의 마지막 릴리스와 관련된 모든 리소스 및 릴리스 내역을 제거하여
나중에 다시 사용할 수 있도록 해제한다.

실제 삭제를 수행하지 않고 어떤 릴리스가 삭제될지 확인하려면
'--dry-run' 플래그를 사용한다.


```
helm uninstall RELEASE_NAME [...] [flags]
```
