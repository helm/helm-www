차트를 원격 저장소로 푸시

### 개요


차트를 레지스트리에 업로드한다.

차트에 연관된 출처(provenance) 파일이 있으면
함께 업로드한다.


```
helm push [chart] [remote] [flags]
```

### 옵션

```
      --ca-file string             이 CA 번들을 사용하여 HTTPS 활성화된 서버의 인증서를 확인

  -h, --help                       push 에 대한 도움말
      --insecure-skip-tls-verify   차트 업로드 시 TLS 인증서 확인 생략
      --key-file string            이 SSL 키 파일을 사용하여 레지스트리 클라이언트 식별
      --password string            요청된 차트를 찾을 수 있는 차트 저장소 비밀번호
      --plain-http                 차트 업로드에 비보안 HTTP 연결 사용
      --username string            요청된 차트를 찾을 수 있는 차트 저장소 사용자 이름
```
