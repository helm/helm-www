릴리스를 이전 리비젼으로 롤백한다.

### 개요


이 명령은 릴리스를 이전 리비젼으로 롤백한다.

rollback 명령어의 첫 번째 인수는 릴리스의 이름이고,
두 번째 인수는 리비젼(버전) 번호이다.
이 인수를 생략하거나 0으로 설정하면 바로 이전 릴리스로 롤백된다.

리비젼 번호를 보려면 'helm history RELEASE'를 실행한다.


```
helm rollback <RELEASE> [REVISION] [flags]
```
