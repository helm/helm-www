헬름 템플릿 언어는 강한 타입을 지원하는 Go 프로그래밍 언어로 구현되어 있습니다.
이 때문에 템플릿 변수에도 _타입(type)_이 있습니다.
대부분의 경우 변수는 다음 자료형 중 하나로 제공됩니다.

- string: 텍스트 문자열
- bool: `true` 또는 `false`
- int: 정수 값 (8, 16, 32, 64 비트의 부호가 있거나(signed) 없는(unsigned) 다양한 자료형이 있습니다)
- float64: 64비트 부동 소수점 값 (8, 16, 32 비트의 다양한 자료형이 있습니다)
- byte slice(`[]byte`), 흔히 (잠재적으로) 바이너리 데이터를 담기 위해 사용됩니다.
- struct(구조체): 속성과 메서드로 이루어진 객체입니다.
- 위의 자료형 중 하나에 대한 슬라이스(인덱스 있는 리스트)
- 위의 자료형 중 하나에 대한 문자열-키 맵(`map[string]interface{}`)

Go에는 이 밖에도 여러 자료형이 있어, 템플릿에서 자료형을 변환해야 할 때가 있습니다.
객체의 자료형을 확인하는 가장 쉬운 방법은 템플릿에서 `printf "%T"`로 타입을 출력하는 것입니다.
`typeOf`와 `kindOf` 함수도 참고하세요.
