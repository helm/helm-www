지정한 경로의 플러그인이 서명되었고 유효한지 검증합니다.

### 개요

이 명령은 Helm 플러그인에 유효한 provenance 파일이 있는지와, 해당 파일이 신뢰할 수 있는 PGP 키로 서명되었는지를 검증합니다.

다음 두 형식을 지원합니다:

- 플러그인 tarball(`.tgz` 또는 `.tar.gz` 파일)
- 설치된 플러그인 디렉터리

설치된 플러그인은 `helm env HELM_PLUGINS`에 표시된 경로 뒤에 플러그인 이름을 붙여 지정합니다. 예:

  helm plugin verify ~/.local/share/helm/plugins/example-cli

`helm plugin package --sign` 명령으로 서명된 플러그인을 만듭니다.

```
helm plugin verify [PATH] [flags]
```
