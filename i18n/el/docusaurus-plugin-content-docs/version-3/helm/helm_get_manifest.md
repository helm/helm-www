κατεβάζει το manifest για ένα δεδομένο release

### Σύνοψη {#synopsis}

Αυτή η εντολή ανακτά το παραγόμενο manifest για ένα δεδομένο release.

Ένα manifest είναι μια αναπαράσταση σε μορφή YAML των πόρων Kubernetes που
δημιουργήθηκαν από το chart (ή τα charts) αυτού του release. Αν ένα chart εξαρτάται από άλλα
charts, αυτοί οι πόροι θα συμπεριληφθούν επίσης στο manifest.


```
helm get manifest RELEASE_NAME [flags]
```
