# Поради щодо використання чартів

Цей посібник охоплює найкращі практики створення чартів, які команда Helm вважає оптимальними. Основна увага приділяється структурі чартів.

Ми зосереджуємося переважно на найкращих практиках для чартів, які можуть бути публічно розгорнуті. Ми розуміємо, що багато чартів призначені лише для внутрішнього використання, і автори таких чартів можуть вважати, що їхні внутрішні інтереси мають пріоритет над нашими пропозиціями, викладеними тут.
