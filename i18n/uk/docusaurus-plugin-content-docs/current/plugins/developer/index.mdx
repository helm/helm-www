Для ознайомлення з концепціями втулків Helm, а також з тим, як структурувати та налаштовувати втулки, ознайомтеся з [Оглядом втулків](/plugins/overview.md).

Цей розділ присвячений розробці втулків Helm — він містить [посібники](#tutorials), інструкції, довідники та додаткову інформацію для розробників щодо створення втулків Helm.

:::info
Вступ до кодової бази системи втулків буде розміщено на сторінці Втулки в розділі [Go SDK](/sdk/index.mdx) документації, яка зʼявиться найближчим часом!
:::
