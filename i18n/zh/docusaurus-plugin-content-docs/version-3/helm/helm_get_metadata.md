获取指定 release 的元数据

### 简介

该命令用于获取指定 release 的元数据信息。


```
helm get metadata RELEASE_NAME [flags]
```

### 可选项

```
  -h, --help            metadata 的帮助信息
  -o, --output format   以指定格式打印输出。允许的值：table、json、yaml（默认 table）
      --revision int    指定 release 的修订版本
```
