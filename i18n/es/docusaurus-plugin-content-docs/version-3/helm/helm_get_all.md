descarga toda la información de un release especificado

### Sinopsis


Este comando muestra información legible sobre las notas, hooks, valores
suministrados y el archivo de manifiesto generado de un release dado.


```
helm get all RELEASE_NAME [flags]
```

### Opciones

```
  -h, --help              ayuda para all
      --revision int      obtiene el release especificado con la revisión indicada
      --template string   plantilla de Go para formatear la salida, ej: {{.Release.Name}}
```
