busca una palabra clave en charts

### Sinopsis

Permite buscar charts de Helm en los distintos lugares donde pueden estar almacenados, incluyendo Artifact Hub y los repositorios que haya añadido. Use los subcomandos de search para buscar en diferentes ubicaciones.

### Opciones

```
  -h, --help   ayuda para search
```
