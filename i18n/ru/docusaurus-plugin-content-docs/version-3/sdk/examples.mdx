В этом документе представлен ряд примеров использования Helm SDK.
Цель — продемонстрировать различные функциональные возможности SDK.

Финальный пример показывает драйвер `main.go` ([ссылка](#driver)), который запускает приведённые ниже действия и включает необходимые вспомогательные функции.

Код примеров находится в директории [helm/helm-www/sdkexamples/](https://github.com/helm/helm-www/tree/main/sdkexamples).
Он полностью функционален.
