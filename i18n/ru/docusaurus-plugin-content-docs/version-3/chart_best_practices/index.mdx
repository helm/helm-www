# Руководство по лучшим практикам для чартов

Это руководство описывает лучшие практики создания чартов, рекомендованные командой Helm.
Оно фокусируется на том, как должны быть структурированы чарты.

Мы уделяем основное внимание лучшим практикам для чартов, которые могут быть публично развёрнуты.
Мы понимаем, что многие чарты предназначены только для внутреннего использования, и авторы таких чартов
могут обнаружить, что их внутренние требования расходятся с нашими рекомендациями.
