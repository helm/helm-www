# Руководство разработчика шаблонов чартов

Это руководство представляет введение в шаблоны чартов Helm с акцентом на
язык шаблонов.

Шаблоны генерируют файлы манифестов — описания ресурсов в формате YAML,
которые понимает Kubernetes. Мы рассмотрим, как структурированы шаблоны,
как их можно использовать, как писать шаблоны Go и как отлаживать
вашу работу.

Это руководство фокусируется на следующих концепциях:

- Язык шаблонов Helm
- Использование значений (values)
- Техники работы с шаблонами

Это руководство ориентировано на изучение тонкостей языка шаблонов Helm.
Другие руководства предоставляют вводные материалы, примеры и лучшие
практики.
