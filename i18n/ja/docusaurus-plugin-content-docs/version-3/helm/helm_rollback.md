release を以前のリビジョンにロールバックします

### 概要

このコマンドは release を以前のリビジョンにロールバックします。

rollback コマンドの第一引数は release 名、第二引数はリビジョン（バージョン）番号です。第二引数を省略するか 0 を指定すると、直前の release にロールバックします。

リビジョン番号を確認するには `helm history RELEASE` を実行してください。


```
helm rollback <RELEASE> [REVISION] [flags]
```
