chart の問題点を検査します

### 概要

このコマンドは、chart のパスを受け取り、一連のテストを実行して chart が正しく構成されているか検証します。

インストールに失敗する原因となる問題を検出した場合は、[ERROR] メッセージを出力します。規約や推奨事項に反する問題を検出した場合は、[WARNING] メッセージを出力します。


```
helm lint PATH [flags]
```
